swap_horiz Migrate from Setlist Helper to Band Central

Band Central is the next generation of Setlist Helper, with cloud sync, band collaboration, and a modern interface. This guide walks you through importing your existing Setlist Helper data into Band Central.

checklist Before You Start

Before beginning the migration, make sure you have the following ready:

  • A Dropbox account (free tier is fine)
  • If you have document or lyrics files: place them all in a single Dropbox folder
  • If you have audio files (backing tracks, rehearsal recordings): place them in the same Dropbox folder

format_list_numbered Step-by-Step Migration

  1. Create an Account

    Sign up or log in at bandcentral.com using your email or Google account.

  2. Create a Band

    From the dashboard, create a new band. This will be the workspace where your imported data lives.

  3. Import Your Data

    On the band card, tap the "Import" button. Follow the prompts to connect your Dropbox account and select the folder containing your files.

verified After Migration

  • Verify that all your songs and setlists appear in Band Central
  • Check that lyrics and documents are attached to the correct songs
  • Confirm audio files are playable
  • Invite your band members to join using the band code

build Troubleshooting

  • Import button not showing? Make sure you are the band owner. Only band owners can import data.
  • Files not importing? Verify that all files are in a single Dropbox folder and that you've granted Band Central access to Dropbox.
  • Missing songs or setlists? Try running the import again. If the issue persists, contact support.
