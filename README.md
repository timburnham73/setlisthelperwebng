# Band Central

From the creator of **Setlist Helper**, Band Central is the next-generation web app for bands and musicians.

Built with modern technologies:
- **Angular 18** + **Material UI** (Angular Material) for a clean, responsive frontend
- **Firebase** for authentication, hosting, and backend services
- **Firestore** as the NoSQL document database for fast, scalable data storage

### Features
- Secure user authentication via Firebase Authentication
- Real-time data syncing with Firestore
- Fully deployed and hosted on Firebase Hosting

⚠️ The app is currently in **beta** — expect rapid improvements and new features!

Feedback and contributions are very welcome!
